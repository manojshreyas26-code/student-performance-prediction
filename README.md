# Student Performance Prediction System

This project predicts student performance using Machine Learning.

## Project Title
Machine Learning Based Student Performance Prediction System

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Description
This project uses Linear Regression to predict student final score based on:
- hours studied
- attendance
- previous score
- gender
- study time
